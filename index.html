Como Fiz Meu Site com Inteligência Artificial

Um guia passo a passo sobre como utilizei IA para criar minha página web de forma rápida e eficiente

Passo a Passo

1. Definição do Objetivo

No início do projeto, deixei claro para a inteligência artificial exatamente o que eu quería desenvolver.

Utilizei prompts detalhados, estratégicos e bem estruturados para explicar como deveria ser o site, qual seria seu objetivo e como a experiência visual deveria funcionar.

Quanto mais específico eu fui nas instruções, melhores foram os resultados gerados pela IA.

Essa etapa foi essential para garantir que o projeto tivesse identidade própria e seguisse exatamente a ideia que eu imaginava.

2. Geração do Código

Depois de definir o objetivo do projeto, escolhi quais linguagens seriam utilizadas na construção do site.

Decidi desenvolver toda a estrutura utilizando HTML, CSS e JavaScript.

Com base nas instruções fornecidas, a IA ajudou na criação do código da página, incluindo:

  • Estrutura do site em HTML
  • Estilização e design com CSS
  • Funcionalidades e interações usando JavaScript

Essa etapa acelerou muito o desenvolvimento e facilitou a criação de um layout moderno e responsivo.

3. Revisão e Ajustes

Após a geração do código, revisei cuidadosamente toda a estrutura do projeto.

Procurei possíveis erros, organizei partes do código e analisei elementos visuais como cores, espaçamentos e aparência geral do site.

Além disso, utilizei esse processo para aprender mais sobre programação, desenvolvimento web e também sobre como a inteligência artificial funciona durante a criação de projetos.

Essa etapa foi importante não apenas para melhorar o site, mas também para aumentar meu conhecimento técnico.

4. Publicação

Com o projeto finalizado, comecei a procurar maneiras gratuitas de publicar o site na internet para que outras pessoas pudessem acessá-lo.

Foi então que descobri o GitHub e a ferramenta GitHub Pages, que me permitiu visualizar o site funcionando diretamente na web e também hospedar o projeto gratuitamente.

Além da hospedagem, consegui gerar um link público para compartilhar o site no meu perfil do Instagram e mostrar o resultado final para qualquer pessoa interessada em conhecer o projeto.

Ferramentas de IA Utilizadas

Perplexity AI - Assistente que gerou o código HTML completo com explicações detalhadas e boas práticas de desenvolvimento web.

Vantagens de Usar IA

  • ⚡ Velocidade: Código gerado em segundos
  • 🎯 Qualidade: Código limpo e bem estruturado
  • 📚 Aprendizado: Entendi como cada parte funciona
  • 💡 Criatividade: Ideias novas para melhorar o design
