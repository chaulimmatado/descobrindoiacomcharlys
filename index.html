Como Fiz Meu Site com IA

Um guia passo a passo sobre como utilizei IA para criar minha página web de forma rápida e eficiente

Passo a Passo

1. Definição do Objetivo

No início do projeto, deixei claro para a inteligência artificial exatamente o que eu queria desenvolver.

Utilizei prompts detalhados, estratégicos e bem estruturados para explicar como deveria ser o site, qual seria seu objetivo e como a experiência visual deveria funcionar.

Quanto mais específico eu fui nas instruções, melhores foram os resultados gerados pela IA.

Essa etapa foi essencial para garantir que o projeto tivesse identidade própria e seguisse exatamente a ideia que eu imaginava.

2. Geração do Código

Depois de definir o objetivo do projeto, escolhi quais linguagens seriam utilizadas na construção do site.

Decidi desenvolver toda a estrutura utilizando HTML, CSS e JavaScript.

  • Estrutura do site em HTML.
  • Estilização e design com CSS.
  • Funcionalidades e interações usando JavaScript.

Essa etapa acelerou muito o desenvolvimento e facilitou a criação de um layout moderno e responsivo.

3. Revisão e Ajustes

Após a geração do código, revisei cuidadosamente toda a estrutura do projeto.

Procurei possíveis erros, organizei partes do código e analisei elementos visuais como cores, espaçamentos e aparência geral do site.

Essa etapa foi importante não apenas para melhorar o site, mas também para aumentar meu conhecimento técnico.

4. Publicação

Com o projeto finalizado, publiquei o site na internet usando o GitHub Pages.

Isso permitiu compartilhar o projeto de forma gratuita e prática com outras pessoas.

O endereço final ficou acessível como um site estático hospedado no GitHub Pages.

Ferramentas de IA Utilizadas

Perplexity AI - Assistente usado para estruturar, revisar e explicar o código com foco em boas práticas.
GitHub Pages - Hospedagem gratuita para publicar seu site estático na internet.

Vantagens de Usar IA

  • ⚡ Velocidade: Código gerado em segundos.
  • 🎯 Qualidade: Estrutura organizada e mais limpa.
  • 📚 Aprendizado: Entender como cada parte funciona.
  • 💡 Criatividade: Novas ideias para melhorar o design.
  • 🔧 Praticidade: Tudo em um único projeto.
  • 🌐 Gratuito: Sem custos de hospedagem.
