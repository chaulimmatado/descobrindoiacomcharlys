Como fiz meu site
com IA

Um guia direto sobre como usei inteligência artificial para criar minha página web — do zero ao ar, sem enrolação.

C

Olá, sou Charlys 👋

Aprendi a criar sites usando IA como ferramenta de desenvolvimento. Aqui documento o processo que usei para criar esta página — do prompt inicial até a publicação no GitHub Pages.

Processo

Passo a Passo

01

Definição do Objetivo

▼

No início do projeto, deixei claro para a IA exatamente o que eu queria. Usei prompts detalhados explicando como deveria ser o site, qual seria seu objetivo e como a experiência visual deveria funcionar.

Quanto mais específico fui nas instruções, melhores foram os resultados. Essa etapa foi essencial para garantir que o projeto tivesse identidade própria.

Exemplo de prompt que usei Crie um site em HTML, CSS e JavaScript com tema dark/gamer que explique como usar IA para criar páginas web. O site deve ter: accordion interativo, alternância de tema claro/escuro, formulário de contato e design responsivo.
02

Geração do Código

▼

Com o objetivo definido, a IA gerou toda a estrutura do projeto usando HTML, CSS e JavaScript.

  • HTML — estrutura e semântica da página
  • CSS — estilização, cores, layout responsivo e animações
  • JavaScript — interações, accordion e alternância de tema

Esse processo que levaria dias foi concluído em minutos, com código organizado e comentado.

03

Revisão e Ajustes

▼

Após a geração do código, revisei cuidadosamente toda a estrutura. Procurei possíveis erros, organizei partes do código e analisei elementos visuais como cores, espaçamentos e aparência geral.

Quando algo não ficava como eu queria, voltava à IA com instruções mais específicas — como "deixa o botão maior" ou "muda essa cor para azul". É um processo iterativo.

04

Publicação no GitHub Pages

▼

Com o projeto finalizado, publiquei gratuitamente usando o GitHub Pages. O processo é simples:

  • Crie um repositório no GitHub
  • Envie seu arquivo index.html
  • Vá em Settings → Pages e ative a publicação
  • Seu site fica disponível em seu-usuario.github.io/nome-do-repo
